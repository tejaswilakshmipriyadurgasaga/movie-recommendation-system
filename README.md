# Movie Recommendation System

## Overview
This project recommends movies using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Algorithm
- Collaborative Filtering
- KNN
- Cosine Similarity

## Dataset
MovieLens Dataset

## Features
- Recommend similar movies
- User-item matrix
- Data visualization

## Project Workflow
1. Load Dataset
2. Data Cleaning
3. Create Pivot Table
4. Train Model
5. Recommend Movies

## Output
Example:
Toy Story → Finding Nemo, Monsters Inc
